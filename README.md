# 🥗 Diet Enhancer

## 📌 Overview
Diet Enhancer is a beginner-friendly data analytics project that helps users track their health and diet.  
It calculates BMI, suggests diet plans, stores progress in a SQL database, exports reports to Excel, and predicts future weight using machine learning.

---

## 🚀 Features
- ✅ BMI Calculation
- ✅ Diet Recommendations
- ✅ SQL Database Storage (SQLite)
- ✅ Excel Report Generation
- ✅ Data Visualization (Graphs)
- ✅ Machine Learning Prediction

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- SQLite (SQL)
- Excel
- Power BI (for dashboard)
